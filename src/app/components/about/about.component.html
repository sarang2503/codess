CODESS : Angular Firebase Code Sharing Platform

Description

This project is a code sharing platform built with Angular and Firebase. It provides users with the ability to contribute and view code snippets. Users must log in to contribute new code snippets, while anonymous users can browse existing contributions. The platform features user authentication, including sign up and login functionality. Each user can create their own code "bin" to contribute and manage their code snippets.

Features

  • User Authentication: Secure user registration and login functionality.
  • Code Contribution: Registered users can contribute new code snippets to the platform.
  • Code Viewing: Anonymous users can view existing code snippets contributed by registered users.
  • Personal Code Bins: Each user has their own space to manage and contribute code snippets.
  • Responsive Design: Optimized for various screen sizes and devices.
